# Performance Analyzer

Скрипт для анализа эффективности работы разработчиков на основе CSV-файлов.

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python main.py --files file1.csv file2.csv --report performance
```

## Параметры:
 - files - пути к CSV-файлам с данными (может быть несколько)
 - report - тип отчета (на данный момент поддерживается только 'performance')
